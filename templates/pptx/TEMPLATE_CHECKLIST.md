- [ ] File saved in correct location

- [ ] Slide size is 16:9 (10" × 5.625")
- [ ] All placeholders spelled correctly
- [ ] Colors match website (use hex codes)
- [ ] Fonts are correct
- [ ] Text boxes are large enough

- [ ] Design matches website pixel-perfect
