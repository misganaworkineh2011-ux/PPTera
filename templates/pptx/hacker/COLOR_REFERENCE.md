# Hacker Theme - Color Reference

Use these exact colors when creating Hacker theme templates in PowerPoint.

## Primary Colors

### Background
- **Very Dark Gray**: `#0d0d0d` (RGB: 13, 13, 13)
- **Dark Gray**: `#141414` (RGB: 20, 20, 20)
- **Medium Dark**: `#1a1a1a` (RGB: 26, 26, 26)

### Primary Green (Terminal)
- **Matrix Green**: `#00ff41` (RGB: 0, 255, 65)
- **Bright Green**: `#39ff14` (RGB: 57, 255, 20)
- **Muted Green**: `#00d435` (RGB: 0, 212, 53)

### Accent Colors
- **Cyan**: `#00d4ff` (RGB: 0, 212, 255)
- **Yellow**: `#ffff00` (RGB: 255, 255, 0)
- **White**: `#ffffff` (RGB: 255, 255, 255)

## Text Colors

### Headings
- **Main Title**: `#39ff14` (Bright Green)
- **Subtitle**: `#00ff41` (Matrix Green)

### Body Text
- **Primary**: `#39ff14` (Bright Green)
- **Secondary**: `#00d435` (Muted Green)
- **Muted**: `#00ff41` at 60% opacity

### Special Text
- **Command**: `#39ff14` (Bright Green)
- **Path**: `#00d4ff` (Cyan)
- **User**: `#00ff41` (Matrix Green)
- **Prompt**: `#ffffff` (White)

## Border & Effects

### Borders
- **Primary Border**: `#00ff41` at 60% opacity
- **Accent Border**: `#00ff41` at 40% opacity
- **Glow Effect**: `#00ff41` at 30% opacity

### Scanlines
- **Scanline Color**: `#00ff41` at 5% opacity
- **Line Height**: 2-3px
- **Spacing**: 4-6px between lines

## Traffic Lights (Terminal Window)

### Window Controls
- **Red (Close)**: `#FF5F56` (RGB: 255, 95, 86)
- **Yellow (Minimize)**: `#FFBD2E` (RGB: 255, 189, 46)
- **Green (Maximize)**: `#27CA40` (RGB: 39, 202, 64)

## Cursor

### Terminal Cursor
- **Color**: `#00ff41` (Matrix Green)
- **Size**: 0.15" × 0.2"
- **Style**: Solid rectangle

## Fonts

### Primary Font
- **Name**: Consolas
- **Fallback**: Courier New, Monaco, monospace

### Font Sizes
- **Title**: 32-36pt, Bold
- **Subtitle**: 18-24pt
- **Body**: 14-16pt
- **Command**: 12-14pt
- **Small**: 10-12pt

## PowerPoint Color Setup

### How to Add Custom Colors

1. **Select Text or Shape**
2. **Click Color Dropdown**
3. **Choose "More Colors"**
4. **Select "Custom" Tab**
5. **Choose "RGB" Color Model**
6. **Enter RGB Values**:
   - Example: For `#00ff41`
   - Red: 0
   - Green: 255
   - Blue: 65
7. **Click OK**

### Save Colors for Reuse

1. **Right-click the color**
2. **Select "Add to Recent Colors"**
3. **Or add to Theme Colors**

## Testing Colors

### Verify Colors Match
1. Take screenshot of website
2. Use color picker tool
3. Compare with PowerPoint colors
4. Adjust if needed

### Color Picker Tools
- PowerPoint built-in eyedropper
- Windows: Paint, Snipping Tool
- Mac: Digital Color Meter
- Online: imagecolorpicker.com

## Tips

1. **Use Hex Codes**: Most accurate
2. **Save Swatches**: Save time on future templates
3. **Test on Dark Background**: Ensure visibility
4. **Check Contrast**: Text must be readable
5. **Consistent Opacity**: Use same transparency values

## Common Mistakes

❌ **Wrong**: Using `#00ff00` (pure green)
✅ **Right**: Using `#00ff41` (matrix green)

❌ **Wrong**: Using Arial font
✅ **Right**: Using Consolas font

❌ **Wrong**: 100% opacity borders
✅ **Right**: 60% opacity borders

❌ **Wrong**: Bright white background
✅ **Right**: Very dark gray `#0d0d0d`

## Need Help?

If colors don't look right:
1. Check RGB values exactly
2. Verify color mode is RGB (not CMYK)
3. Check monitor brightness
4. Compare with website screenshot
5. Test export to verify
